# Home_and_Lifestyle
This project presents an interactive Home and Lifestyle Sales Dashboard built using Power BI to analyze sales performance and business trends. The dashboard provides insights into revenue, profit, customer behavior, and regional performance.

It includes data cleaning, transformation, and modeling to create meaningful KPIs and visual reports. Advanced visuals such as bar charts, line charts, pie charts, and slicers are used to enhance data exploration and decision-making.

The dashboard helps stakeholders monitor sales growth, identify top-performing products, and understand customer purchasing patterns. It supports data-driven strategies to improve profitability and operational efficiency.
